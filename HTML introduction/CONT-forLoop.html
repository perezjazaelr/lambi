PHP - For Loop

Description

 A program by default follows a sequential execution of statements.    If the program flow is directed towards any of earlier statements    in the program, it constitutes a loop. The for statement in PHP is a    convenient tool to constitute a loop in a PHP script.

Syntax

1.<?php

2.// Using a for loop to print numbers from 1 to 10

3.

4.echo "Counting from 1 to 5 using a for loop: ";

5.

6.for ($i= 1; $i <= 5; $i++) {

7.echo $i, " ";

8. }

9.

10.?>
