What are Operators in PHP?

As in any programming language, PHP also has operators which are symbols (sometimes keywords) that are predefined to perform certain commonly required operations on one or more operands.

We have the following types of operators in PHP:

● Arithmetic Operator

● Comparison Operator

● Logical Operator

● Assignment Operator

● String Operator

● Array Operator

● Conditional (or Ternary Operator)

Syntax

1.<?php

2.$a = 52;

3.$b = 10;

4.$c = $a + $b;

5.echo "Addition Operation Result: $c \n"; // output 62

6.$c = $a - $b;

7.echo "Subtraction Operation Result: $c \n"; // output 42

8.$c = $a * $b;

9.echo "Multiplication Operation Result: $c \n"; // output 520

10.$c = $a / $b;

11.echo "Division Operation Result: $c \n"; // output 5.2

12.$c = $a % $b;

13.echo "Modulus Operation Result: $c \n"; // output 2

14.$c = $a++;

15.echo "Increment Operation Result: $c \n"; // output 52

16.$c = $a--;

17.echo "Decrement Operation Result: $c"; // output 53

18.?>
